1. Розгортування реалізованої програмної системи.
Для розгортання системи використаємо сервіс https://render.com/, що надає послуги для безоплатного розгортування невеликих сервісів.
Створимо API додаток, підʼєднавши GitHub репозиторій, увівши потрібні команди для запуску та додавши конфігураційні змінні. Панель керування додатком представлена на рисунку 1.
